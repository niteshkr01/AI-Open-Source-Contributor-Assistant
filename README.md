## AI Open Source Contributor Assistant



Most developers want to contribute to open source but struggle to find the right projects and understand complex issues. This tool solves that.



Search any GitHub repository, explore its issues, filter beginner-friendly ones, and get AI-powered explanations — all in one place.



## Live Demo

\[ai-open-source-contributor-assistan.vercel.app](https://ai-open-source-contributor-assistan.vercel.app)



## Features

\- Search any public GitHub repository by name

\- View real-time repository data — stars, language, description

\- Browse open GitHub issues with state indicators

\- Filter issues by \*\*Good First Issue\*\* and \*\*Help Wanted\*\* labels

\- Get AI-generated explanations for any issue in plain English



## 🛠️ Tech Stack

| | Tech |
|--|------|
| Frontend | React · TypeScript · Vite |
| Backend | Node.js · Express |
| AI | Groq · Llama 3.3 70B |
| API | GitHub REST |
| Deploy | Vercel |



## Architecture

React Frontend → Express Backend → Groq AI + GitHub REST API
